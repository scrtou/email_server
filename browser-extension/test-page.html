测试说明： 这个页面包含多种类型的登录和注册表单，用于测试浏览器插件的自动检测功能。

测试步骤：

  1. 确保已安装并配置好 Email Server 浏览器插件
  2. 在插件中登录您的 Email Server 账户
  3. 填写下面的任意表单并提交
  4. 观察插件是否显示保存提示
  5. 确认账号信息是否正确保存到 Email Server

预期行为：
  • 提交表单后，页面右上角应该出现插件的保存提示
  • 提示框应该显示检测到的平台名称（test-page.html 或当前域名）
  • 提示框应该显示检测到的邮箱地址和用户名
